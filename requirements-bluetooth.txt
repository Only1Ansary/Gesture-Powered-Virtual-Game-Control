# ============================================================
#  Optional: in-app Bluetooth scanning for the admin screen.
#
#  Install AFTER the main requirements:
#      pip install -r requirements-bluetooth.txt
#
#  This enables bluetooth_admin.py to scan for your admin
#  device at runtime.  You can skip this if you use the
#  static MAC / name approach (see README § Admin Bluetooth).
#
#  Platform notes
#  --------------
#  Linux / macOS   — pybluez usually installs cleanly.
#  Windows         — PyBluez often fails to build from PyPI.
#                    Options:
#                      • Use list_bluetooth_devices.ps1 to get
#                        your device MAC and set admin_bluetooth_mac
#                        in config.json (no PyBluez needed).
#                      • Try a pre-built wheel for your Python
#                        version from an unofficial mirror.
#                      • Or set admin_bluetooth_force: true in
#                        config.json for local testing only.
# ============================================================
